# ThirdPersonCrosshair
This mod allows you to display crosshairs in third person view, just like in **Minecraft 1.8.9** and earlier.

# Features
- Displays the crosshair in **third-person view**
- Almost the same behavior as **Minecraft 1.8.9 and earlier**
- No key bindings
- No configuration files
- Lightweight and safe

# Supported Versions
Currently it only works with **1.12.2 Forge**, but we plan to port it to other Forge versions in the future.
<br>There are absolutely no plans to add support for **Fabric**.

Please don't ask when it will be ported.<br>
Alternatively, anyone can contribute by adding support for a new version and submitting a pull request.

# Build 
_You need [gradle](https://gradle.org/) to build this mod._

Build command:
```bash
cd versions/<version>
gradlew build
````

The built mod will be located in:
```
build/libs/
```
